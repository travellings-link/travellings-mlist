# 开往成员列表

这是 [开往成员列表](https://list.travellings.cn/) 的前端源代码仓库。

Author: [xuanzhi33](https://github.com/xuanzhi33)

## 功能

- 向公众开放
    - 网站信息展示
    - 巡查信息展示
    - 搜索、筛选
    - 举报网站

- 仅维护组成员
    - 编辑成员信息
    - 从 Github Issues 同步新成员

## 技术栈

- 框架：[Vue.js](https://vuejs.org/)
- 构建工具：[Vite](https://vitejs.dev/)

## 调试

```bash
pnpm install
pnpm dev
```

## 部署

```bash
pnpm build
```
